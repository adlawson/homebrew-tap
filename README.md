# Andrew's Homebrew Tap

## Installing

```shell
# install from github
brew install adlawson/tap/<cask or formula name>

# or build from a clone
brew install --build-from-source /path/to/adlawson/homebrew-tap/<Cask|Formula>/<cask or formula name>.rb
```

## Documentation

`brew help`, `man brew` or check [Homebrew's documentation](https://docs.brew.sh).
